MULTILINE PROMPT FORMATTING GUIDE
=================================

METHOD 1: BLOCK DELIMITERS (RECOMMENDED FOR LONG PROMPTS)
=========================================================

---PROMPT-START---
You are an expert document analyst specializing in creating concise and informative summaries.
Your task is to create a comprehensive summary of the document text below.

CRITICAL SYSTEM INSTRUCTIONS:
1. You MUST respond with ONLY a valid JSON object - NO OTHER TEXT
2. Do NOT include any explanatory text, markdown, or other formatting

Please provide:
- A brief executive summary (2-3 sentences)
- Key findings or main points (bullet format)
- Any recommendations or action items
- Important dates, numbers, or metrics mentioned

Format your response as a JSON object with these fields:
{
  "executive_summary": "...",
  "key_findings": ["...", "..."],
  "recommendations": ["...", "..."],
  "metrics": {"key": "value"}
}
---PROMPT-END---
---DATA-START---
your_actual_base64_data_here
---DATA-END---


METHOD 2: MULTILINE PREFIX (GOOD FOR MEDIUM PROMPTS)
====================================================

prompt-multiline:You are an expert document analyst.
Please create a comprehensive analysis including:

1. Executive summary (2-3 sentences)
2. Key findings and main points
3. Recommendations or action items
4. Important metrics and dates

Please respond in JSON format with the following structure:
{
  "executive_summary": "...",
  "key_findings": ["...", "..."],
  "recommendations": ["...", "..."],
  "metrics": {}
}

data:your_actual_base64_data_here


METHOD 3: TRADITIONAL SINGLE LINE (FOR SHORT PROMPTS)
=====================================================

prompt:Please analyze this document and provide a JSON summary with executive_summary, key_findings, recommendations, and metrics fields
data:your_actual_base64_data_here
